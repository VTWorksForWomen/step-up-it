Accessing items in list

myFriends[2] # 'Chuck'

Dictionaries

Lists with specific indicies

myDictionary = {'index1':'value1','index2':'value2'}

myFriendsAges = {'Alice':40,'Bob':20,'Chuck':75}
weekdayNames = {'sun':'Sunday','mon':'Monday','tue':'Tuesday','wed':'Wednesday','thu':'Thursday','fri':'Friday','sat':'Saturday'}

Accessing items in dictionary

myFriendsAges['Bob'] # 20
myFriendsAges['Shirley'] # ERROR!
myFriendsAges.get('Chuck') # 75
myFriendsAges.get('Shirley') # Nothing/False

Loops

Repeat logic for specified iterations

"for" and "while"

for month in monthsOfYear:
    print(month)
while myAge < 30:
    myAge = myAge + 1
    print(myAge)

Read From an Existing File

Take contents of a file and use them in your script

file = open('filename.txt','r')
with open('filename.txt','r') as file:
    print(len(file.read()))
Caveats
  • Permissions on files can make this tricky and frustrating
  • Different ways to locate the file
    • Absolute path: start with root of filesystem
      • Relative path: start with location of python script and navigate relative to it
