Boolean Variables

Boolean variables represent the logical values True and False


var taskIsComplete = true;
var taskIsDeleted = false;

If you try to use another variable as a boolean, JavaScript will guess. The number 0, the empty string '', undefined, and null are considered false, everything else reads as true.

The if statement

Use if to decide which lines of code to execute, based on a condition.


var taskIsComplete = true;
if (taskIsComplete) {
  alert('Its done!'); // This code runs
}
				

var numberOfTasks = 5;
if (numberOfTasks === 0) {
  console.log('You finished all your tasks'); // Doesnt run
}

Comparison Operators

Example Name Result
5 == "5" Equal true
5 === "5" Identical false
0 != false Not equal false
0 !== false Not identical true
5 < 10 Less than true
5 > 10 Greater than false
"a" <= "b" Less than or equal to true
"a" >= "b" Greater than or equal to false

Watch out!

Don't mix up = and ==


var numberOfTasks = 5;
if (numberOfTasks = 10) {
    alert('There are 10 tasks'); // This will always run
}

The if/else statement

Use else to provide an alternate set of instructions.


var numberOfTasks = 5;
if (numberOfTasks === 0) {
  console.log('You finished all your tasks');
} else {
  console.log('Back to work!');
}

The if/else if/else statement

If you have multiple conditions, you can use else if.


var numberOfTasks = 5;
if (numberOfTasks === 0) {
  console.log('You finished all your tasks');
} else if (numberOfTasks === 1) {
  console.log('Only one more to go!');
} else {
  console.log('Back to work!');
}

Using logical operators

You can use these operators to combine conditions.


var taskIsComplete = false;
var numberOfRemainingTasks = 5;
if (taskIsComplete && numberOfRemainingTasks === 0) {
    console.log('You just finished your last task!');
} else {
    console.log('You still have some tasks left');
}

For loops

For loops allow you to execute a defined number of times.


//will count 1 to 10
for (var i = 1; i <= 10; i++) {
    console.log(i); 
}

For loops include 3 things: A starting value (i = 1), a conditional that determines whether the loop should run before each iteration (i < 10), and some code that should be run at the end of each iteration (i++).

Loops and logic

You can add other statements or logical operators inside the loops.


//Count from 1 to 50
for (var i = 1; i <= 50; i++) {
    console.log(i);       
    //Says 'Buzz' after multiples of three
    if (i % 3 == 0) {
        console.log(' Buzz');
    }        
    //Says 'Bang' after multiples of five
    if (i % 5 == 0) {
        console.log(' Bang');
    }
}

Break

To exit a loop, use the break statement.


//Count from 100 to 200              
for (var i = 100; i <= 200; i++) {
    console.log('Testing ' + i);
    //Stop at the first multiple of 7
    if (i % 7 == 0) {
        console.log('Found it! ' + i);
        break;
    }
}

Arrays

Arrays are ordered lists of values.


var tasks = ['Groceries', 'Wash the dog', 'Learn to program'];

You can put different types of data into an array.


var colors = ['Red', 'Orange', 'Yellow', 'Green'];
var lotteryNumbers = [33, 72, 64, 18, 17, 85];
var myFavoriteThings = ['Broccoli', 1024, 'Sherlock'];

Array Length

The length property tells you how many things are in an array


var tasks = ['Groceries', 'Wash the dog', 'Learn to program'];
console.log('There are ' + tasks.length ' tasks');

Using Arrays

You can access items with "bracket notation" by using the position of the object you want. Computers start counting at zero.


var tasks = ['Groceries', 'Wash the dog', 'Learn to program'];
var firstTask = tasks[0];
var lastTask = tasks[2];

Changing arrays

You can use bracket notation to change an item in an array


var tasks = ['Groceries', 'Wash the dog', 'Learn to program'];
tasks[0] = 'Get some sleep';

Add items to arrays

You can use "push" to add a value to an array.


var tasks = ['Get some sleep', 'Wash the dog', 'Learn to program'];
tasks.push('Call your mum');

Iterating through arrays

Use a for loop to easily process each item in an array.


var tasks = ['Get some sleep', 'Wash the dog', 'Learn to program'];
for (var i = 0; i < tasks.length; i++) {
  console.log(tasks[i]);
}
